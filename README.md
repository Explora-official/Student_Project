# Student CS Chatbot

A comprehensive AI-powered chatbot for computer science education, featuring specialized agents for different CS domains.

## Features

- **Multi-Agent System**: Specialized AI agents for DSA, Operating Systems, and Networking
- **Real-time Chat**: WebSocket-based real-time messaging
- **LangGraph Workflows**: Advanced AI decision-making workflows
- **Modern Tech Stack**: Node.js backend, React frontend, MongoDB, Redis

## Project Structure

See [PROJECT_STRUCTURE_GUIDE.md](Plan_and_docs/PROJECT_STRUCTURE_GUIDE.md) for detailed folder structure.

## Getting Started

1. Clone the repository
2. Set up the backend (see backend/README.md)
3. Set up the frontend (see frontend/README.md)
4. Run with Docker Compose

## Technologies Used

- Backend: Node.js, Express, TypeScript, LangChain, LangGraph
- Frontend: React, TypeScript, Vite, Tailwind CSS
- Database: MongoDB
- Cache: Redis
- AI: Claude/GPT models
